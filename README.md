# xPlat_PMToolKit
AI-Powered PM Toolkit that assists teams with preparing for meetings and creating next steps automatically afterwards. Simply provide the transcript and off you go!

Configurable LLM data isolation with three tiers: Tier 1 (cloud API with zero-retention), Tier 2 (pre-processing redaction of sensitive identifiers), Tier 3 (self-hosted/private LLM for air-gapped processing). Swappable per project or workspace.
